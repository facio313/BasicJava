git / github

1. github에 회원가입하고 로그인한다.
2. git을 다운로드하고 설치한다.
3. github에서 새 repository를 만든다.
4. github/repository에 나와있는 것들을 복사한다.

새로운 저장소와 폴더 간 연결 시
폴더에서 오른쪽 클릭하고 Git Bash Here 누른 다음에
새 repository 만들 때 나온 것들을 복사해서 Git Bash Here에 붙여넣는다.
(*아니면 명령 프롬프터를 켜서 "cd 경로"  // 현재 경로 변경하고 밑에 거 수행하면 된다)
	echo "# gitTest" >> README.md
	git init
	git add README.md
	git commit -m "first commit"
	git branch -M main
	git remote add origin https://github.com/facio313/gitTest.git // 그대로 복사 말고 경로 확인!
	git push -u origin main

5. 새로운 파일이 만들어졌을 때!
	git status // 확인용
	git add. // add --all
	git commit -m "first commit" // 현재 상태 저장
	git push -u origin main // 올려주기


==> 하면 저장소에 올라감!!!

(6.블로그 푸쉬
	git add --all
	git commit -m "Initial commit" //이거 안 돼서 5번에 커밋함
	git push -u origin main)
